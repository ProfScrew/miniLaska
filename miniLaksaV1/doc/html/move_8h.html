◆ movement

typedef struct availableMovement movement

Contains list of available movement of a player.

It uses two 2d arrays(pointer to pointer). Where one dimention is fixed on either 6 when capture/eat or 4 when normal move, the other dimention value is stored in the corresponding count value.
